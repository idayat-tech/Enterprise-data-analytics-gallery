# 🔒 Enterprise Shadow AI Risk & Data Leakage Audit

### 📁 Project Overview
Designed an enterprise cybersecurity audit solution that detects unauthorized AI tool usage ("Shadow AI"), identifies potential data leakage risks, and monitors compliance with internal AI governance policies.

The project simulates firewall logs from multiple departments, automatically cleans inconsistent data using SQL Views, classifies sanctioned vs. unsanctioned AI vendors, assigns risk levels through a SQL-based scoring engine, and visualizes security incidents in Power BI.

### 📊 Executive Dashboard
![Enterprise Shadow AI Command Center](enterprise-shadow-ai-audit/shadow-ai-dashboard.png)

### 🔍 Business Problems Solved
- Detect employees using unauthorized AI tools
- Monitor confidential data uploaded to external AI platforms
- Identify departments with the highest security risk
- Help security teams prioritize critical incidents

### 📈 Key Insights
- Total Shadow AI policy violations
- Total data uploaded to unauthorized AI platforms (MB)
- Critical Level 5 security incidents
- Department-level data leakage analysis
- Most frequently used unauthorized AI platforms
- Detailed investigation table for security audits

### 🛠️ Technologies
MySQL • SQL Views • CASE Statements • LEFT JOIN • Power BI • DAX

# 📈 Product Growth Analytics & User Cohort Retention

### 📁 Project Overview
Built a SaaS product analytics dashboard to measure user retention, feature adoption, and customer engagement across weekly cohorts.

Using SQL for cohort preparation and Power BI for interactive analytics, the dashboard enables product managers to identify onboarding issues, compare acquisition channels, monitor feature adoption, and detect users at risk of churn.

# 🌍 Global Cross-Border Payroll & Currency Automation

### 📁 Project Overview
Developed an international payroll reporting solution that automates cross-border salary conversion, standardizes multi-currency payroll reporting, and enables secure regional reporting using Row-Level Security (RLS).

The solution reduces manual spreadsheet calculations while giving management a consolidated view of global payroll costs across different countries and currencies.

### 📊 Dashboard Preview
![Global Payroll Dashboard](global-cross-border-payroll/payroll-dashboard.png)

### 🔍 Business Problems Solved
- Eliminate manual currency conversion
- Consolidate international payroll reporting
- Monitor payroll costs across regions
- Restrict payroll visibility using role-based security

### 📈 Key Insights
- Payroll by Country
- Payroll by Currency
- Currency Conversion Summary
- Regional Payroll Distribution
- Department Payroll Breakdown
- Secure Role-Level Reporting (RLS)

### 🛠️ Technologies
SQL • Power BI • DAX • Row-Level Security (RLS) • Currency Conversion
